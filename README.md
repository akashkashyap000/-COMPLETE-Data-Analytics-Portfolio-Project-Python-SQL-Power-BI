🛍️ Customer Shopping Behavior Analysis
📌 Project Overview

This project focuses on analyzing customer shopping behavior to help a retail company improve sales, customer satisfaction, and long-term loyalty. The analysis uncovers patterns across demographics, product categories, and sales channels (online vs offline), and identifies key factors influencing purchasing decisions.

🎯 Business Problem

A retail company wants to understand:
What drives customer purchasing decisions?
How do factors like discounts, reviews, seasons, and payment methods affect sales?
How can customer data be used to improve engagement and marketing strategies?

Goal:
Leverage data to identify trends, optimize product strategies, and enhance customer experience.

🧰 Tech Stack
Python → Data Cleaning & Preprocessing
SQL (MySQL/PostgreSQL) → Data Analysis & Querying
Power BI → Dashboard & Visualization
Excel/CSV → Data Source

🔧 Project Workflow

2️⃣ Data Analysis (SQL)

Repeat customer behavior
3️⃣ Data Visualization (Power BI)
Interactive dashboards
Sales trends over time

4️⃣ Reporting
Insights summary

Strategy suggestions

📊 Key Insights
Discounts significantly influence purchase decisions
Online sales are growing faster than offline
Certain product categories generate higher repeat purchases
Customer reviews impact buying behavior
Seasonal trends affect sales patterns
